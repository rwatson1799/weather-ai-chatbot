# weather-ai-chatbot

A small weather AI chatbot completed by following these tutorials from the Vercel website.

https://vercel.com/kb/guide/weather-api-with-express

https://vercel.com/kb/guide/mcp-server-with-weather-tool-express

---

To develop locally:

```
npm install
vc dev
```

```
open http://localhost:3000
```

To build locally:

```
npm install
vc build
```

To deploy:

```
npm install
vc deploy
```
